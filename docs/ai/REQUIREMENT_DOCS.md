# Client Documentation Demands

Supplied by the client in addition to the coding challenge in README.md. The submission must include:

1. Full prompt transcript: the raw conversation log with the AI tool(s) used, exported in its entirety.
2. An annotated diff or decision log, marking inline or in a separate document:
   - Where AI output was used directly.
   - Where AI output was modified and why.
   - Where AI output was rejected and why.
3. Verification information: how it was confirmed that the AI-generated pieces work.
   - What tests were used?
   - Which edge cases were tested?
4. Tools and task mappings: how AI was deployed differently across the problem and which tools were used for each task.
5. A short self-critique:
   - If you could change part of the solution, what would you change and why?
   - Which part of the solution seems strongest, and why?
   - Which part of the solution seems weakest, and why?
